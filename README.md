Sistema Camedda Veículos
O Sistema Camedda Veículos é uma aplicação Java desenvolvida para o gerenciamento de veículos de uma locadora, permitindo realizar operações de Create, Read, Update e Delete (CRUD) com persistência em banco de dados relacional. O sistema utiliza PostgreSQL, acessado via JDBC, e foi estruturado seguindo o padrão de arquitetura MVC (Model-View-Controller). As operações de inserção e exclusão de registros são automaticamente registradas no arquivo de auditoria transacoes.csv.
